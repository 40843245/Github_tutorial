# Objectives
How to look at files of your repo in File Explorer via GitHub Desktop?
# method
## method 1:
Step 1:
Open GitHub Desktop.
Step 2:
Select your repo.
Step 3:
Navigate Changes.
Step 4:
Click "Show in Explorer" button.

Step 5:
That's done.

## method 2:
Select Repository->Show in Explorer

## method 3:
Click the key shortcut "Ctrl+Shify+F".
