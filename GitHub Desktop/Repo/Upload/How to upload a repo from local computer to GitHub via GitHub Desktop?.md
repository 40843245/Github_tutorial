# Objectives
How to upload a repo from local computer to GitHub via GitHub Desktop?
# Step:
Step 1:
Open GitHub Desktop.
Step 2:
Select the repo.
Step 3:
Select Changes.
Step 4:
Click "Publish repository" button.
Step 5:
Ensure the information.
Step 6:
Click "Publish repository" button.
Step 7:
That's done.
