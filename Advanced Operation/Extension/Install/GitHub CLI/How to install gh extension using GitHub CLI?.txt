How to install gh extensions using GitHub CLI?
-----------------------------------------
Step 1:
Open Terminal.
Step 2:
Check login. If NOT login, please login first.
Step 3:
In terminal, type
gh extension install <REPO-NAME> [option]
where
The argument [option] can be 
--pin <string>
which refers pin extension to a release tag or commit ref

-----------------------------------------
Ref:
https://cli.github.com/manual/gh_extension_install
