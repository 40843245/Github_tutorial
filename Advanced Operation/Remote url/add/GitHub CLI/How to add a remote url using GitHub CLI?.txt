How to add a remote url using GitHub CLI?
----------------------------------------
Step 1:
Open terminal.
Step 2:
Check login. If NOT login, please login first.
Step 3:
In terminal, type
git remote add <URL-NAME> <GIT>
where 
The argument <URL-NAME> refers the name of remote url.
And the argument <GIT> refers a .git file which the url points to.
In this example,
git remote add origin https://github.com/40843245/test4.git
the name of url is "origin", and
the url points to a .git file "https://github.com/40843245/test4.git"
